Noir / UltraHonk benchmarks

This tool benchmarks the time it takes to generate UltraHonk proofs in the browser for various circuit sizes and standard library black-box functions. The circuits are the same as those benchmarked in noir-benchmarks.

nargo version 1.0.0-beta.0 was used to compile the circuits.

The file sizes of the noir-lang and bb dependencies are as follows. They are downloaded in the background and are necessary for proof generation. The compiled Noir circuits, however, are much smaller than these files.

  • noirc_abi_wasm_bg.wasm: 1.66 MB
  • acvm_js_bg.wasm: 2.01 MB
  • @aztec/bb.js: 75.08 MB

Glossary:

  • Circuit: The number of circuit constraints (a power of 2) or the name of the circuit which uses a standard library function.
  • Circuit size: The uncompressed size of the compiled Noir program (which is a JSON file).
  • Witness computation: The time taken to generate the witness using noir.execute().
  • Proof generation: The time taken to generate the proof.
  • Proof valid?: Whether the proof is valid.
